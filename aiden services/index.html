Guide to Cheating


General Vocabulary

FP: Facepunch (the owners/devs of the game)
EAC: Easy Anti-Cheat (anti cheat for Rust)
CERB: Cerberus (usually referred for 3/7 day bans)
LAV: Last Activity Viewer (program used for searching PCs)
Exit Scam: A basic scam where someone reputable ends up quitting what they sell and scams you or other people.
Inject: Loading the cheat's code into the games code
Emulation: Mimics and edits the files sent back and forth to EAC
Desync: Desynchronization has your local player doing something separate from what the server is getting sent back
HWID: Short for hardware ID. You have a bound hardware ID for the components of your PC. These are used by EAC to identify you with accounts that you play on. Some simple information that you would need to actively clear and spoof are: mac address, disk serials, registry, wmi, and KUSER_SHARED_DATA.ProcessorFeatures (0xFFFFF78000000274).

Chapter 1, Basics of Cheating

Some of the most basic parts of cheating is to know the cheat you are using, the server you are on, and the features you are using. An example of why this is important is because some servers like Rusty Moose (official) have server sided anti cheats that pick up commands like noclip, debug camera, and features like magic bullet, hitbox override, and more. These detections all added up will end up automatically banning you from the server. Some of the most basic and important features that you should use while cheating (rage) are: Aimbot/Silent Aim, ESP, omnisprint, magic bullet, desync (weapon), bullet thickness, instant eoka, and hit override. Some of these features may not work on some servers due to anti cheats, but they will help you instantly kill people with AK, Eoka, and other guns. Some of these features are self explanatory, but some may be harder for people to know about if they are new to cheating. Some users of cheats use some features that are extremely blatant to all servers that have the simple plugins of anticheats and such. An example of this is that they will dig up stashes by just standing on them and looking down, like anyone would regularly do to their own stashes. What people don’t know at times, is that some servers like AndysolAM/RustAcademy have a plugin called “Stash Traps” which would notify admins inside of a Discord channel using webhook/announcements saying “Username had opened a auto generated stash trap (Number of times they opened one)”. More plugins also include implementations of RustAdmin/Battlemetrics, which would auto kick you for having a certain Steam level, real hour count of playtime on Rust, or your account being banned on a server that has shared ban lists with other servers. An example of this is that if you get banned on AndysolAM, you will get banned automatically on RustAcademy. This is because they have shared ban lists, and are also owned by the same owners. Some servers are smart to cheat on, while others are not smart to cheat on. For example, if you cheat on a Facepunch server, using certain features or you reach a high amount of reports on it, you will get a 3 day ban quite fast. These servers used to be a lot easier to cheat on, but they had recently had a better EAC update that had modified the severity/threshold that you could have on them.
